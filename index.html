Business Recommendations
🔴 Priority 1 — Immediate
Deploy LightGBM in Shadow Mode
Run alongside existing rule-based system. Compare outputs for 2 weeks before full cutover. Target <100ms inference latency.
🔴 Priority 2 — Immediate
Set Threshold to 0.30 (Not 0.50)
Default 0.50 misses 30%+ of fraud. At $856K avg fraud value vs $50 investigation cost, lower threshold has 17,000:1 ROI asymmetry.
🟡 Priority 3 — Month 1
SHAP Explanation Service
Build API endpoint returning top-3 SHAP features for each flagged transaction. Required for analyst review UI and regulatory compliance.
🟡 Priority 4 — Month 1
Weekly Model Monitoring
Track ROC-AUC on recent transactions. Alert if drops >0.02. Monitor feature drift (KS test). Trigger retraining if score falls below 0.97.
🟢 Priority 5 — Month 2
Retrain on Full History
Current model trained on 20% sample. Full 6.3M transaction training expected to push ROC-AUC to 0.999+. Schedule monthly retraining.
🟢 Priority 6 — Month 3
Graph Network Features
Model transaction networks between accounts. Fraud rings have distinct graph signatures. Expected to add 0.5–1% AUC improvement.
